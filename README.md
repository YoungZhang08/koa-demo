# koa-demo(基于koa2的简单注册登录)
### Node+Koa2+Mysql2

## 数据库

### 1. 登录
`
mysql -u root -p
`

### 2. 创建数据库
`
create database koa_demo;
`

### 3. 使用数据库
`
use koa_demo;
`

### 4. 建表(略)
- id（逐渐递增）
- name（用户名）
- pass（用户密码）

## 项目启动

`
git  clone https://github.com/YoungZhang08/koa-demo.git
`

`
cd koa-demo
`

`
cnpm i
`

`
nodemon index.js
`
